# Plan Maestro v2

Construimos un PMS nuevo, mas optimo que CQR. CQR queda como checklist operativo para no olvidar procesos, campos sensibles ni reportes necesarios.

## Principios

- Organizar por flujos reales: reservar, alojar, cobrar, facturar, limpiar, mantener y reportar.
- Menos pantallas y menos duplicacion.
- Permisos fuertes para acciones sensibles.
- Auditoria obligatoria.
- Web + mobile como PWA.
- AFIP real antes del go-live.

## Primer sprint

`login -> dashboard -> habitaciones -> cambio de estado -> auditoria`.
